さまざまなユーティリティを提供します． {@link vavi.util.Debug} クラスは大変便利です．

デバッグ手法

  • デバッグ情報を表示する

    Debug.println(...);

  • 既存のクラスをオーバライドする

    JTree tree= new JTree() {

    public void addTreeSelectionListener(TreeSelectionListener l) {
        super.addTreeSelectionListener(l);
Debug.println(Debug.DEBUG, ...);
    }
}

  • 現在デバッグしているところだけを表示する

    Debug.println(Lebel.SPECIAL, ...);

% java -Ddebug.lebel=0 foo ...

  • 呼び出しメソッドを表示する

    Debug.println(... Debug.getCallerMethod() ...);

Debug.println(... Debug.getCallerMethod(3) ...);

Debug.println(... Debug.getTopCallerMethod("vavi") ...);

  • デバッグコードを取り除く
